This is a python app that uses the camera to detect hands, facial expressions and objects using google mediapipe, you can also finger gun your camera to control the mouse cursor

Features:
- Face Tracking
- Hand Tracking
- Body Tracking
- Object Recognition
- Mouse Pointer With Fingers
- Left Click By Opening Mouth
- Adjust Volume By Pinching And Draging Up And Down

Requires these python libraries:
Mediapipe, Opencv, PyAutoGui, Numpy, Psutil and Pyvirtcam

pip install opencv-python mediapipe numpy pyautogui psutil pyvirtualcam
